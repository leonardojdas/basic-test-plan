Test Cases
Instructions attempting to illuminate if the function works properly

Test case ID: EM-001-Pass

Description
The average of the marks is greater than required minimum, and all individual marks are greater than the minimal specified.
Inputs
Marks: [55, 90, 70], minAverage: 60, minIndividual: 50
Expected result
"PASS"

Test case ID: EM-002-LowAverage

Description
All individual marks are greater than the minimal stated, but the average is less than the minimum average specified.
Inputs
Marks: [60, 55, 60], minAverage: 60, minIndividual: 50
Expected result
"FAIL: The average too low"

Test case ID: EM-003-LowIndividualMark

Description
The average of the marks is above the required minimum, but one of the individual marks is below the individual minimum.
Inputs
Marks: [49, 80, 80], minAverage: 60, minIndividual: 50
Expected result
"FAIL: individual mark too low"
